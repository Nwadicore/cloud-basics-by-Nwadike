## What I Learned:
- Cloud computing is the delivery of computing services over the internet.
- The main types of cloud services are:

  - PaaS (Platform as a Service)

- Types of cloud deployment:
  - Public Cloud
  - Private Cloud
  - Hybrid Cloud

## Why Cloud?
- Cost savings
- Scalability
- Flexibility
- Fast deployment
- Global access
